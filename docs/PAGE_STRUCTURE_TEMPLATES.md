## Section Psychology Framework

Every section must serve ONE of these purposes:

| Purpose | What It Does | Example Sections |
|---------|--------------|------------------|
| **STOP** | Stop scrolling, capture attention | Hero |
| **TRUST** | Build credibility immediately | Trust Bar, Certifications |
| **SHOW** | Demonstrate capability/offering | Services, Portfolio |
| **PROVE** | Provide social proof | Testimonials, Case Studies |
| **EXPLAIN** | Reduce friction, educate | Process, How It Works |
| **OVERCOME** | Handle objections | FAQ, Guarantees |
| **CONVERT** | Drive action | CTA sections |
| **CONNECT** | Enable contact | Contact, Footer |

## Mobile Optimization Rules

### Sticky Elements by Page Type

| Page Type | Sticky Element | Trigger |
|-----------|----------------|---------|
| Homepage | Phone FAB | After hero scroll |
| Service Page | Funnel trigger | After benefits |
| Emergency Page | Emergency banner | Always visible |

### Section Reordering for Mobile

Some sections may need different order on mobile:

```
DESKTOP:                          MOBILE:
1. Hero                           1. Hero (simplified)
2. Services                       2. CTA (moved up!)
3. Benefits                       3. Services
4. Process                        4. Benefits (collapsed)
5. Testimonials                   5. Testimonials (carousel)
6. CTA                            6. FAQ (accordion)
7. FAQ                            7. Footer
8. Footer
```

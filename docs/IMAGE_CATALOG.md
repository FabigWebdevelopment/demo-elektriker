## How to Use This Catalog

1. **Find the image** by filename
2. **Read the description** to know what it actually shows
3. **Check "Use For"** to see if it fits your need
4. **Check "NOT For"** to avoid mismatches
5. **If no image fits** → Generate a new one (see bottom)

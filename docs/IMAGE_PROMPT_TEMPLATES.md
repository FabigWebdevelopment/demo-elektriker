# Image Prompt Templates

## Overview

This document provides standardized, copy-paste-ready prompt templates for generating consistent, high-quality images across landing pages and service pages.

**Strategy:** All images feature the CEO/owner as a consistent persona, using a reference image to maintain visual identity across the entire website.

---

## Reference Image System

### How It Works

1. **Reference Image:** Upload a clear photo of the business owner (Thomas Müller)
2. **Consistency:** Every generated image uses this reference for facial features
3. **Variations:** Same person in different contexts (consulting, working, team photos)

### Reference Image Requirements

```
- Clear, well-lit portrait photo
- Face clearly visible (front or 3/4 view)
- Neutral expression
- Resolution: minimum 1024x1024
- Format: JPG or PNG
```

## Usage Instructions

### Step 1: Prepare Reference Image

Upload a clear headshot of the business owner to use as reference for facial consistency.

### Step 2: Choose Template

Select the appropriate template based on:
- **Landing page:** Hero, Consulting, At Work, Team
- **Service page:** Choose service-specific template

### Step 3: Customize Details

Replace placeholders:
- `[Company Name]` → Actual company name
- `[City]` → Actual city
- `[Specific brands]` → Brands they actually use

### Step 4: Generate

Use Google Imagen 3 or similar with:
- Reference image uploaded first
- Full prompt from template
- Resolution: 1536x1024 (landscape) or 1024x1536 (portrait)

### Step 5: Review Checklist

Before using generated image:
- [ ] Face matches reference image
- [ ] No wire color errors (Brown=L, Blue=N, Green-Yellow=PE)
- [ ] Branded products are correct/realistic
- [ ] No safety violations visible
- [ ] Professional quality (not stock photo feel)
- [ ] German cultural elements present

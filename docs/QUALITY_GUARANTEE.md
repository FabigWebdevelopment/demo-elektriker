## Gate 1: Strategy Review

### Checklist

**Market Intelligence:**
- [ ] Top 5 competitors analyzed
- [ ] Market gaps identified
- [ ] Positioning angles defined
- [ ] Trust signals researched

**Customer Psychology:**
- [ ] 3 buyer personas created
- [ ] Pain points mapped (functional, emotional, social)
- [ ] Desires mapped (functional, emotional, social)
- [ ] Objections identified with overrides
- [ ] Decision journey documented

**SEO Strategy:**
- [ ] Primary keyword selected (volume + intent)
- [ ] Secondary keywords (5-10)
- [ ] Long-tail keywords (10-20)
- [ ] Content architecture planned
- [ ] Internal linking strategy defined

**Funnel Strategy:**
- [ ] Funnel type selected
- [ ] Page section architecture defined
- [ ] Conversion path mapped
- [ ] Friction points identified
- [ ] Trust sequence planned

### Gate 1 Pass Criteria

| Item | Required |
|------|----------|
| Competitive analysis | Complete with 5 competitors |
| Buyer personas | 3 personas with full mapping |
| Keyword strategy | Primary + 5 secondary minimum |
| Funnel architecture | All sections with psychological purpose |

## Continuous Improvement

### After Every Project

**Retrospective Questions:**
1. What worked well?
2. What could improve?
3. Any new patterns discovered?
4. Any agent prompts need updating?

**Update Process:**
1. Document learnings
2. Update relevant agent prompts
3. Add to checklist if new issue type
4. Train team on improvements

### Monthly Review

**Aggregate Metrics:**
- Average SEO score
- Average conversion score
- Most common issues
- Time per project

**Improvement Actions:**
- Refine prompts for common issues
- Add automation for repetitive checks
- Update documentation
- Share learnings
